 1. Project Overview

    

    Number Guessing Game
    A simple, fun number guessing game built for beginners.
    The app picks a random number between 1 and 100, and your job is to guess it.
    

 
3. Technologies Used

   

    a-- HTML5
    b-- CSS3
    c-- JavaScript (vanilla JS)
    d-- Git & GitHub


5. Instructions to Run the Project

   

   Step 1: Clone the Repository Open your terminal or
                command prompt and run the following command: 
                git clone https://github.com/ankiitsony-git-hub/number-guessing-game-/tree/main
                This will download the project files to your computer.
   

   Step 2: Open the Project Folder
   
   Step 3: Open the folder in a code editor such as Visual Studio Code.
   
   Step 4: Run the Application : Simply open the index.html file in your web browser.
   
   Step 5: Use the guessing number game
